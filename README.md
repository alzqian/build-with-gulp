# **build-with-gulp**
> 用gulp构建web项目的个人练习 
### 主文件是 gulpfile.js

## **Usage**  

* 压缩且合并所有js文件  
`$ gulp js-to-all`
* 压缩且合并所有css文件  
`$ gulp css-to-all`
* 分别压缩每个js文件  
`$ gulp js`
* 分别压缩每个css文件  
`$ gulp css`
* 压缩图片文件  
`$ gulp img`
* js + css+ img   
`$ gulp entire`
* 监听 js 和 css   
`$ gulp auto`
* entire + auto   
`$ gulp smart`
* 压缩图片输入与输出路径相同，会替换原文件   
`$ gulp img-same-folder`
